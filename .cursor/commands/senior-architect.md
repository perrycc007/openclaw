# Senior Architect

Comprehensive software architecture skill for designing scalable, maintainable systems. Includes architecture diagram generation, system design patterns, tech stack decision frameworks, and dependency analysis.

## When to Use

- Designing system architecture
- Making technical decisions
- Creating architecture diagrams
- Evaluating trade-offs
- Defining integration patterns

## Capabilities

### 1. Architecture Diagram Generator

```bash
python3 .cursor/skills/senior-architect/scripts/architecture_diagram_generator.py <project-path> [--verbose]
```

### 2. Project Architect

```bash
python3 .cursor/skills/senior-architect/scripts/project_architect.py <target-path> [--verbose]
```

### 3. Dependency Analyzer

```bash
python3 .cursor/skills/senior-architect/scripts/dependency_analyzer.py <target> [--json] [--output file.json]
```
